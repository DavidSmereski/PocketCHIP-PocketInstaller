Getting Started

Interactive fiction is a literary format that the reader interacts with and directs. As the reader you'll type commands in natural language, like "open door" or "unlock chest". Many frequently used commands can often be abbreviated; instead of typing "go north" one can instead type "n" for example. Likewise all the other directions, "i" for "inventory" and "g" for "again" are commonplace. Look in the above references to get a lot more information on playing interactive fiction in general.

Zoom provides a menu to browse through installed stories and play them. Use the cursor keys to go through the list and the enter key to select one.

Extensions

This PocketInstaller copy of Zoom includes a handful of sample stories that you can play immediately. More stories can be downloaded from the Interactive Fiction Archive linked above. Not all available formats can be played with Zoom; look specifically for Z-Machine files.
