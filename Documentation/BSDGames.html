Getting Started

These are the classic games included with every distribution of BSD Unix and derivatives and available as a package for most Linux distributions. It includes several text-based games made to run in a terminal, including: adventure, arithmetic, atc, backgammon, battlestar, bcd, boggle, caesar, canfield, countmail, cribbage, dab, go-fish, gomoku, hack, hangman, hunt, mille, monop, morse, number, pig, phantasia, pom, ppt, primes, quiz, random, rain, robots, rogue, rot13, sail, snake, tetris, trek, wargames, worm, worms, wump, and wtf. Note that even though these are referred to as "games", some are actually text processors (for example, morse converts text to Morse code, and rot13 encodes and decodes using the ROT13 cipher) or math utilities (for example, primes outputs ascending prime numbers, and number outputs the equivalent English text for a given number) or file utilities (for example, random outputs random lines from a file). Some of the games included are individually quite famous (for example trek, wumpus, adventure, and tetris). The game rogue is the original source of the gaming adjective "roguelike".

PocketC.H.I.P. Considerations

Every one of these is designed to be run from within a terminal, so on the PocketC.H.I.P. each of these can be run by invoking the PocketC.H.I.P. Terminal app and typing the name of the desired program. Help on individual programs can be obtained via the manual (man) command, also typed within the terminal. So for example to get help on using hangman or more information about it, type man hangman in the Terminal.

Extensions

In some cases friendlier (and/or non-text-only) versions of these games are available in other packages. In particular adventure has friendlier alternatives that can run within Zoom, Gargoyle, or Lectrote, and Tetris has multiple graphical options.
