Getting Started

Interactive fiction is a literary format that the reader interacts with and directs. As the reader you'll type commands in natural language, like "open door" or "unlock chest". Many frequently used commands can often be abbreviated; instead of typing "go north" one can instead type "n" for example. Likewise all the other directions, "i" for "inventory" and "g" for "again" are commonplace. Look in the above references to get a lot more information on playing interactive fiction in general.

Lectrote provides a file browser to peruse installed stories and play them. Use the touch screen or cursor keys to browse to /usr/local/share/IF and go through the list and press the Open button to start one. You may find that this is quite a bit easier if you use a stylus or the keyboard.

One of the most useful keys when using Lectrote is the F11 key. It will enable and disable fullscreen mode.

Extensions

This PocketInstaller copy of Lectrote includes a handful of sample stories that you can play immediately. More stories can be downloaded from the Interactive Fiction Archive linked above. Not all available formats can be played with Lectrote; look specifically for Z-Machine, Glulx, and Hugo files.
